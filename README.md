# Balanced-Physics-Scene
<h1>Introduccion</h1>
En este proyecto veremos algunas de las funciones de unity en cuanto a las colisiones, gravedades y movimiento del personaje controlado por el usuario.
Se usarán conocimientos previamente utilizados mediante tutoriales.

<h1>Descripcion de mi proyecto</h1>

La idea principal de mi proyecto es crear un minijuego el cual constará de un plano que tenga diferentes obstáculos como esferas que obstaculizan el paso al jugador para llegar hasta el fondo de el plano, dependiendo de en que lugar caiga el jugador sumará mas o menos puntos, el objetivo es conseguir el mayor numero de puntos llegando a los sitios mas dificiles de llegar los cuales se marcan en la parte inferior del plano
<img width="1007" height="555" alt="image" src="https://github.com/user-attachments/assets/777961df-b56c-4cf9-909a-095c1dba7858" />
Utilicé el mesh collider para crear las colisiones de los planos a pesar de tener algunos detalles que me costó arreglar, tambien utilicé el rigid body para que el personaje tuviera gravedad y así pudiera moverse libremente.

<h1>Mecánicas principales</h1>
-Colisiones de personaje con planos y esferas<br>
-Movimiento de personaje
